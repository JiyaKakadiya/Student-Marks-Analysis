# 📊 Student Marks Analysis using Pandas

This project performs basic data analysis on student marks using Python and Pandas.

---

## 🔍 Features

- Handling missing values using mean
- Creating new columns (Total Marks)
- Sorting data based on performance
- Finding subject-wise toppers
- Calculating average marks

---

## 📁 Dataset

The dataset contains marks of students in:
- Maths
- Science
- English

Some values are intentionally missing to demonstrate data cleaning.

---

## ⚙️ Technologies Used

- Python
- Pandas

---

## 🚀 Steps Performed

1. Created a DataFrame
2. Checked for null values
3. Filled missing values with mean
4. Created a "Total Marks" column
5. Sorted students based on total marks
6. Calculated average marks
7. Identified top scorers in each subject
